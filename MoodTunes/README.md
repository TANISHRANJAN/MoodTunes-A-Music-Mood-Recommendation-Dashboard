# MoodTunes - Music Mood Recommendation Dashboard

MoodTunes is a web application that recommends music playlists based on your current mood. The application features animated, color-themed cards for different moods, each linking to curated playlists.

## Features

- Animated mood cards with smooth hover effects
- Color themes representing different moods
- Direct links to mood-based playlists
- Responsive design that works on all devices
- Bootstrap 5 for modern styling
- Animate.css for smooth animations

## Setup

1. Clone this repository
2. Open `index.html` in your web browser
3. Click on any mood card to access the corresponding playlist

## Technical Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Animate.css

## Customization

To add new moods or modify existing ones:

1. Add a new card section in `index.html`
2. Define the mood's color scheme in `css/style.css`
3. Update the playlist links to your preferred music platform

## Note

Replace the placeholder playlist links with actual Spotify/YouTube playlist URLs before deploying.
